# Dyer Auto Group — Review Response Generator

Single-page web app that generates professional responses to customer reviews on Google/Yelp.

## Features

- **Instant response generation** — paste a review, select options, get 3 response variations in seconds
- **Three tones**: Professional, Warm/Personal, and Brief
- **Sentiment-aware**: different templates for positive (4-5★), neutral (3★), and negative (1-2★) reviews
- **33+ unique templates** — randomized selection prevents cookie-cutter responses
- **Variable substitution** — store name, department, customer name, employee mentions, GM contact info
- **Copy to clipboard** — one click to copy any response
- **History** — last 20 generated responses saved in localStorage
- **No API keys needed** — fully self-contained, works offline

## Stores & Departments

- Stores: Dyer Chevy, Dyer Mazda, Dyer Subaru, Dyer Kia
- Departments: Sales, Service, Collision, Parts, Marine

## Usage

Open `index.html` in any browser. No build step or server required.

## Design

Navy (#1a2744) and gold (#c9a84c) Dyer branding. Responsive, works on desktop and mobile.
