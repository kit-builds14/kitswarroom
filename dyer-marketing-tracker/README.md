# Dyer Auto Group — Marketing Campaign Tracker

Dark-themed single-page web app for tracking marketing campaign ROI across all Dyer Auto Group dealerships.

## Features

- **Dashboard** — KPIs, lead funnel, spend/ROI charts by channel, monthly trends
- **Campaign Cards** — Add, edit, archive campaigns with full performance metrics
- **Channel Comparison** — Sortable table with color-coded ROI across all channels
- **CSV Export** — One-click download of all campaign data
- **12 Pre-loaded Campaigns** — Realistic data across Google Ads, Facebook, TV, Radio, Direct Mail, Email, Billboard, Events, and more

## Tech

- Single `index.html` file, no build step
- Chart.js 4.x via CDN
- localStorage persistence
- Dark navy/gold executive theme

## Usage

Open `index.html` in any modern browser. Data persists in localStorage. To reset to sample data, clear localStorage and refresh.
