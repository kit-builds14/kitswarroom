# Dyer Auto Group — People Analyzer

EOS-inspired People Analyzer / Culture Dashboard for Dyer Auto Group.

## Features

- **People Analyzer Grid** — Rate employees on Core Values (+, +/-, -) and GWC (Get it, Want it, Capacity) with click-to-toggle
- **The Bar** — Visual separator showing who's above and below the bar
- **Department Filtering** — Sales, Service, Collision, Parts, F&I, Admin, Marine
- **Summary Dashboard** — Pie chart (people quadrant), department health bar chart, action-needed list
- **Quarterly Reviews** — Snapshot current state, compare quarters, track trends, add notes
- **Editable Core Values** — Pre-populated with Accountability, Integrity, Team First, Customer Obsessed, Growth Mindset
- **CSV Export** — One-click download of all employee data
- **Local Storage** — All data persists in the browser

## Usage

Open `index.html` in any modern browser. No server required.

## Tech

Single HTML file. Chart.js via CDN. Zero dependencies to install.
