# Dyer Auto Group – Digital Business Cards

A single-file web app that generates mobile-friendly digital business cards for Dyer Auto Group employees.

## Features

- **Card Creator** – Name, title, store, phone/cell/email, certifications, tagline, social links
- **Customer-Facing Card** – Clean mobile layout with one-tap Call/Text/Email, vCard download, Google Maps directions, QR code, appointment scheduling, store hours, and Google Review link
- **Card Management** – Create, edit, delete, and share cards via unique URL hash (`#john-smith`)
- **vCard Export** – Downloadable `.vcf` contact file
- **QR Code** – Canvas-generated, no external APIs
- **5 Sample Employees** – Pre-populated across different stores and roles
- **100% Client-Side** – All data in localStorage, no server required

## Stores

Dyer Chevrolet · Mazda · Subaru · Kia · Collision · Marine

## Usage

Open `index.html` in any browser. Share individual card URLs (e.g., `index.html#mike-johnson`) via text message to customers.

## Tech

Single HTML file. No dependencies. No build step. Mobile-first design with Dyer navy/gold branding.
