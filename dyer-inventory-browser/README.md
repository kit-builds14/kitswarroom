# Dyer Auto Group — Vehicle Inventory Browser

Single-page HTML app for browsing Dyer Auto Group's vehicle inventory across all four brands: **Chevrolet, Mazda, Subaru, and Kia**.

## Features

- **Sticky filter bar** — Condition, Make (multi-select), Body Style, Price/Year range sliders, Sort
- **45 pre-populated vehicles** — realistic mix of New, Used, and CPO across all brands
- **Responsive card grid** — mobile-first, brand-colored placeholder images
- **Detail modal** — full specs, features list, similar vehicles carousel
- **Lead capture** — "Get E-Price" form saves to localStorage
- **100% client-side** — no backend, instant filtering, zero dependencies

## Design

- Light theme, navy accents (`#1a2744`), gold CTAs (`#d4a836`)
- Brand-colored vehicle placeholders (blue/red/sky/purple)
- Mobile-responsive grid and filter layout

## Usage

Open `index.html` in any browser. No build step required.
