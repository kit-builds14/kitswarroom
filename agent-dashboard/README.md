# Agent Performance Dashboard

Real-time performance tracking for Kit's multi-agent team.

## Features

- **Today's Stats** — Tasks completed, build time, success rate, LOC
- **Agent Leaderboard** — Per-agent metrics and status
- **Task Timeline** — Gantt-style view of agent work
- **Quality Metrics** — First-pass success, QA rejections, post-ship bugs
- **Sprint Burndown** — Chart.js burndown with ideal vs actual
- **Cost Tracker** — Token usage and cost per agent

## Stack

Single HTML file. Chart.js via CDN for the burndown chart. Dark navy + gold theme.

## Usage

Open `index.html` in any browser. No build step, no server needed.
