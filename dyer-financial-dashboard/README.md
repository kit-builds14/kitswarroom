# Dyer Auto Group — Monthly Financial Summary / P&L Dashboard

Executive-level monthly financial overview for Dyer Auto Group.

## Features

- **Month/Year selector** with instant refresh
- **Top-line summary cards**: Revenue, Gross Profit, Net Profit, Units Sold, Avg Gross/Unit (with MoM delta)
- **Department P&L breakdown** (tabbed): New Vehicles, Used Vehicles, F&I, Service, Parts, Collision, Marine
- **Store comparison**: Chevy vs Mazda vs Subaru vs Kia with best/worst highlighting
- **Trend charts** (Chart.js): 12-month gross profit, department contribution pie, units sold bar, expense ratio
- **Data entry** modal per store/month, CSV import/export, localStorage persistence

## Usage

Open `index.html` in any modern browser. No server required.

Pre-populated with 3 months of realistic data (Nov 2025 – Jan 2026).

## Design

Executive dark theme (navy/gold), print-friendly via `@media print`.
